# Credit Stand-In Decision Engine  
AI Hackathon Project – Rule Engine + FastAPI + Explainability

---

## 📌 Overview  
This project implements a **Stand-In Credit Decision Engine** that evaluates credit applications using a JSON‑driven rule engine.  
It includes:

- FastAPI service  
- Rule engine with externalized rules  
- Failover logic (DMP placeholder)  
- AI explanation layer  
- Simulation engine  
- Synthetic data generator  
- Full test suite  

This structure is designed for clarity, extensibility, and hackathon‑friendly demonstration.
